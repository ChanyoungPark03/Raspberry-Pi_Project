# 스마트 화분 자동 관수 시스템 (Smart Pot)

## 📌 프로젝트 개요  
라즈베리파이 기반으로 토양 수분 센서, DHT11 온습도 센서, 릴레이(물펌프), LCD 디스플레이를 활용한  
자동 물 공급 및 상태 표시 시스템.

---

## 🛠 주요 기능  
- 토양 수분 측정 (Soil Sensor)
- 온도/습도 측정 (DHT11)
- 릴레이를 통한 물펌프 제어
- LCD에 상태 정보 표시
- Flask 웹 서버로 실시간 모니터링
